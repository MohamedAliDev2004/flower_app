# 🌸 FlowerApp
FlowerApp is a robust, full-stack Flutter application powered by Firebase. This project serves as a showcase of my ability to build scalable, real-time applications by leveraging the complete suite of Firebase services.

## ⚡ Powered by Firebase
Firebase Authentication: Secure user management (Email/Password, Google Sign-in).

Cloud Firestore: Real-time NoSQL database for managing orders, products, and user data.

Firebase Storage: Cloud storage for high-quality product images and user assets.


## 🚀 Key Features
Real-time Data: Seamlessly synced data between the app and the cloud using Firestore.

Secure Auth: Protected user journeys with robust authentication flows.

Image Management: Efficient upload and retrieval of media assets via Firebase Storage.

Cloud Logic: Server-side synchronization that ensures data consistency across all devices.

## 🛠 Tech Stack
Framework: Flutter

Language: Dart

Backend as a Service: Firebase (Auth, Firestore, Storage)

State Management: (Provider)

## 💡 Why this project?
This project demonstrates my proficiency in integrating mobile front-end with cloud back-end services, handling real-time data streams, and ensuring security in user-facing applications.
